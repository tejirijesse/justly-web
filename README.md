# Justly — Website

Marketing site for Justly. **Your rights. Secured.**

## Stack
- React 18 + Vite
- Tailwind CSS
- Framer Motion
- lucide-react

## Develop
```bash
npm install
npm run dev
```
Opens on `http://localhost:5173` (or next free port).

## Build
```bash
npm run build
npm run preview
```

## Structure
Single-file React app — all components inline in `src/App.jsx`.
